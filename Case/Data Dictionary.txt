node1 & node2: names of the nodes connected. trail: edge attribute indicating the abbreviated name of the trail for each edge. For example: rs = red square distance: edge attribute indicating trail length in miles. color: trail color used for plotting. estimate: edge attribute indicating whether the edge distance is estimated from eyeballing the trailmap (1=yes, 0=no) as some distances are not provided. This is solely for reference; it is not used for analysis.
